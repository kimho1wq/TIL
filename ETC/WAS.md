# WAS(Web Application Server)

### WAS 란 
- WAS는 클라이언트 요청에 대해 동적인 처리를 담당하는 영역으로, 웹 서버와 달리 애플리케이션 로직을 실행할 수 있도록 구성되어 있다
- WAS는 비지니스 로직의 처리를 위한 DB 연동, 보안, 로깅 등의 기능을 제공하며 Tomcat, JBoss, WebLogic 등이 있다
