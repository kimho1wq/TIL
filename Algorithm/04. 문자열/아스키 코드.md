# [BOJ] [11654. 아스키 코드](https://www.acmicpc.net/problem/11654)

## 1. 문제

알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.


| 시간 제한 | 메모리 제한 |
|:------|:-------| 
| 1 초   | 256 MB |


## 입력

알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.

## 출력

입력으로 주어진 글자의 아스키 코드 값을 출력한다.


### 예제 입력 1

```
A
```

### 예제 출력 1

```
65
```


### 예제 입력 2

```
0
```

### 예제 출력 2

```
48
```


### 예제 입력 3

```
z
```

### 예제 출력 3

```
122
```
